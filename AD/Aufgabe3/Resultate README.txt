##########Installation##########

Einbinden der Adts:

Um eine Adt einzubinden in ein Projekt, klickt man mit der rechten Maustaste auf ein Projekt und wählt Properties.
Unter Java Build Path -> Libraries wählt man Add external Jars und sucht sich die jars aus die man einbinden möchte.


Um das fertige Projekt mit allen Adts und Tests zu öffnen, entpackt man zunächst ad_Aufgabe2.zip.
In Eclipse kann man unter "File -> Import -> General -> Existing Projects into Workspace" das entpackte Projekt importieren.

---------------------------------------------

Öffnen Sie die Resultate Excel Datei.

Klicken Sie nun auf den rechts von der Überschrift platziereten Button.
Klicken Sie sich nun in den Ordner mit dem Projekt, in diesem wurden die .csv erstellt.
(Vorher erstellt mit der "Benchmark" Klasse innerhalb des Projekts).
Nun können Sie eine der gewünschten Excel dateien einfügen.

Mit dem Reset-Button löschen Sie alle von Ihnen eingefügten Daten heraus.
